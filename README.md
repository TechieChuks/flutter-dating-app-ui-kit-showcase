# Flutter Dating App UI Kit

A pixel-perfect Flutter implementation of a modern dating application UI based on the original Figma design:

https://www.figma.com/design/VvOO4MU0mOAzdVxm5y2WTY/Dating-app---Free-UI-Kit?node-id=0-1&p=f

This project showcases:
- 4 Onboarding carousel with parallax animation, Smooth page transitions and a Custom dots indicator
- Authentication screens (Sign In, Create Account) with phone number verfication and an OTP screen
- Profile details Screen that includes a Calnder Screen
- I am a Screen
- Passions Screen
- Friends Screen
- Notification Screen
- Main Screen
- Swipe Left and Right Screen
- Filter Screen 
- Match Screen
- Messages Screen
- Chat Screen
- Stories Screen
- Profile Screen
- Full PhotoScreen

- Reusable UI components (buttons, typography, layout widgets)
- Local and Network asset management for images & icons


This repository contains UI only (no backend, no business logic).
